# PM-Accelerator-Weather-App-Assessment
A web application to record past, present, and forecasted weather for travel purposes. It supports celsius/fahreneheit conversion, containing an archive of high and low temperatures, precipitation, and windspeed for previous years as well as a five-day forecast. It allows the user to save their search information as a readable PDF or as a JSON for dataset creation and programming purposes.
